# 协作 ## 二维火PM分享 ----- # 定义 > 合作或协作（英语：Collaboration）是指一种由两个或两个以上的个人或团体为一个共同的目标而交集在一起共同工作，举例说明：一个知识分子可以透过合作的关系分享知识，再而经过学习和共识达到创作的目标。 > *-- [wikipedia](https://zh.wikipedia.org/wiki/%E5%90%88%E4%BD%9C)* ----- # 团队的力量 在非洲草原上如果看见羚羊在奔跑，那一定是狮子来了；如果看见狮子在躲避，那就是象群发怒了；如果看见成千上完的狮子和大象集体逃命的壮观景象，那么什么来了呢？——是蚂蚁军团来了！ ----- # 什么是团队 - 团队的定义 - 为了共同目标在一起工作的一群人 - 团队的基本要素 - 共同目标 - 互相依赖 - 归宿感 - 责任心 ----- # 高效团队的基本特征（一） - 有明确的共同愿景 > 工作没有远景就会枯燥乏味；有远景而没有实干只是一种空想；有远景再加上实干就成了世界的希望。 - 团队成员对愿景的认同度 - 高效的沟通、良好的合作 ----- # 高效团队的基本特征（二） - 高效的领导 - 以身作则，身先士卒，凡是以团队的利益为重 - 具有较强的协调和激励他人的能力 - 懂得有效授权 - 高素质的员工 - 积极的工作态度 - 具有不同的专业知识、技能和经验 ----- # “我可以，你不行”模式 - 你说可笑不可笑？别人花很长时间办一件事，是慢；我花很长时间办一件事，是仔细。 - 别人省略一些事情，是懒惰；我省略一些事情，是效率。 - 别人若不经吩咐做某一件事，是越权；我这样做，是有主动性。 - 别人强烈坚持观点，是顽固不化；我强烈坚持观点，是坚定、有原则。 - 别人忽略了规则，是不负责任；我忽略规则，是创新。 ----- # 团队协作与沟通 - 沟通是团队协作的基础 - 沟通是相互学习的基础 - 沟通是建立互相信任的基础 - 沟通是建立良好人际关系的基础 ----- # 关于沟通 - 沟通的定义 > 人与人之间通过语言、文字、符号或者其他表现形式进行信息传递和交换的过程 - 沟通的目的 > 控制、指导、激励、决策、反馈和评价，信息交流和社会需求 ----- # 沟通的前提 - 对别人感兴趣 - 尊重他人 - 接受自己 - 自信 - 先理解别人再要求被理解 - 解除误解增强信任 ----- # 沟通的六要素 - 我们做的每件事情都在沟通 - 发出信息的方式总会影响接收信息的方式 - 真正的沟通时信息的被接收 - 开始传递信息的方式决定了我们沟通的结构 - 双向沟通：我们既要收集信息又要分发信息 - 沟通是一种友谊舞 ----- # 有效的沟通技巧 - 有意识使用身体语言 - 小心使用术语 - 坦白承认自己的感觉 - 使用开放型问题 - 耐心倾听 - 找出异议 - 找出反面异议的根源 - 建设性反对 ----- # 沟通的障碍 - 缺乏信息或者知识 - 没有倾听 - 没有完全理解问题和不当询问 - 只顾按照自己的预先设计的思路发展 - 没有理解他人的需要 - 没有经过慎重思考就得出结论 - 失去耐心 - 时间太短 ----- # 沟通中的致命过失 - 傲慢无理：评价安慰、扮演或者标榜为心理学家、讽刺挖苦、过分或者不恰当的寻味 - 发号施令：命令、威胁、多余的劝告 - 回避：模棱两可、保留信息、转移注意力 ----- # 无效沟通的负面影响 - 自尊和自信降低 - 沮丧和敌对 - 部门间矛盾 - 失去创造力 - 失球团队精神 - 错误无效 ----- # 给予正确的信息 - 你的生意是一种乐器 - 为了清晰而强调 - 运用语言、语调、音量和语速达到最好的沟通效果 - 运用第一人称直截了当表达自己的意思 - 选用中性、明确、具体、好记、有活力的词汇 - 多用**请求**,不用命令 - 在没有说明原因之前不要说你喜欢或者不喜欢什么 ----- # 反馈 - 提出反馈意见要具体 - 仅仅表达行为，不要添加解释或标签 - 提供富有建设性的反馈意见 - 反馈意见要有现实性
